# Configure network instance (VRF)

## URL

```
ios-essential:vrfs/vrf={{l3_vpn_bgp_vrf}}
```

## OPENCONFIG YANG

[YANG models](https://github.com/FRINXio/openconfig/tree/master/network-instance/src/main/yang)

```javascript
{
    "vrf": {
        "id": {{l3_vpn_bgp_vrf}}
	"description": {{l3_vpn_bgp_vrf_desc}}
    }
}
```
